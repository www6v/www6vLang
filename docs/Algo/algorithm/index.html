    二叉搜索树: # 

       + 左子树上所有结点的值均小于它的根结点的值； 
     右子树上所有结点的值均大于它的根结点的值； 
     它的左、右子树也分别为二叉搜索树。
   + 最差时间复杂度的二叉搜索树是O(n)，退化成链表;
     所以提出了平衡二叉搜索树， 最差时间复杂度是O(logn)。
     平衡二叉搜索树: Red-Black Tree, Splay Tree, AVL Tree
